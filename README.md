# FB Page Token Extractor (FBPTE)
Python script for extracting an EAAAA token from a Facebook Page.

## Installation
```
git clone https://github.com/kiryudev/FBPTE/
cd FBPTE
python FBPTE.py
```
## How to use
* Execute the script.
* Input your account token. (EAAAA,EAADY)

## Dependencies
- requests
- sys
- os
